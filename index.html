CardioWatch(Detecting Cardiovascular Diseases in PPG signals using Deep Learning)

In my final year project, I developed a deep learning model for the early detection of cardiovascular diseases using photoplethysmography (PPG) signals. I utilized a combination of Convolutional Neural Network (CNN) and Long Short-Term Memory (LSTM) architectures to extract temporal and spatial features from the PPG signals. Furthermore, I integrated the HeartPy module, a powerful tool in the Python ecosystem for heart rate analysis, to derive additional insightful metrics from the PPG data. This enabled the model to analyze intricate patterns and variations in the PPG signals, potentially indicating cardiovascular anomalies. The project underlines my expertise in deep learning, data analysis, and my passion for leveraging AI in the healthcare sector to improve diagnostics and patient outcomes. Implemented Hybrid Sort and Mountain Sort algorithms as part of the Sorting Algorithms Lab Assignment. Hybrid Sort combines Bubble Sort and Selection Sort, arranging the largest element at the end and the lowest element at the beginning of the array. Mountain Sort efficiently sorts integers by storing the first N/2 lowest numbers in ascending order and the next N/2 numbers in descending order.
